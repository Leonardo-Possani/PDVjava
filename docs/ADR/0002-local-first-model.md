# ADR 0002 — Modelo Local-First

## Status
Aceito

## Contexto

Pequenos comércios frequentemente possuem internet instável.

## Decisão

O sistema operará prioritariamente offline.

Validação de licença ocorre periodicamente, sem bloqueio imediato.

## Consequências

- Maior complexidade de sincronização
- Maior robustez operacional
- Independência de conectividade constante
